CX Object System
================

Specs: https://pad.riseup.net/p/cxo

## Packages

- `cmd/cxod` - CXO daemon
- `cmd/cxocli` - cli to control CXO daemon
- `data` - CX obejcts database
- `node` - CXO networking core
- `skyobejct` - CX objects tree
